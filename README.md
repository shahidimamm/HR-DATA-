# HR_DATA_ANALYSIS
Data analytics project showcasing HR Data  analysis using sql and Tableau

 - Data Storytelling | HR Analytics Dashboard
 Overview

This project demonstrates an end-to-end HR Data Analytics workflow, where a raw dataset of 1,470 employee records was transformed into meaningful business insights.

The objective was to analyze workforce trends, understand attrition patterns, and build an interactive dashboard that supports data-driven HR decision-making.

The project covers the full analytics lifecycle:

Data loading

Data cleaning and transformation

SQL analysis using MySQL Server

Interactive Tableau dashboard development

Analytical report creation

Executive presentation using Gamma

This project highlights both technical expertise and data storytelling ability.

 Dataset

Domain: HR Analytics

Total Employees: 1,470

Active Employees: 1,233

Attrition Count: 237

Attrition Rate: 16.12%

The dataset includes employee-level information such as:

Gender

Age

Job roles

Job satisfaction

Attrition status

The goal was to identify key factors influencing employee attrition and engagement.
